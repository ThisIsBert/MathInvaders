# MathInvaders

MathInvaders is a static progressive web app for Android tablets in portrait mode.
It turns multiplication practice into a Space-Invaders-style arcade game.

## Run locally

Open `index.html` directly in a browser, or serve the folder with any static file
server. No build step is required.

## GitHub Pages

Publish the repository root with GitHub Pages. The app uses only static files:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- `sw.js`
- `assets/`

The service worker enables offline caching after the first successful page load.
